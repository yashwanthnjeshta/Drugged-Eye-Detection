# Drugged Eye Detection System

EyeDrug Scan Pro is a Python-based desktop application that detects possible drug influence using eye image analysis.

## Features
- User authentication system
- Eye image upload
- Redness detection
- Pupil size analysis
- Pixel distribution visualization
- Automatic drug detection suggestion
- Report generation

## Technologies Used
Python
OpenCV
Tkinter
NumPy
Matplotlib
SQLite

## How to Run

Install dependencies

pip install -r requirements.txt

Run the program

python DrugEyeDetection.py
